# 🎨 Design System Pinteya E-commerce

Bienvenido al **Design System Pinteya**, un sistema de diseño completo mobile-first especializado en productos de pinturería, ferretería y corralón para el mercado argentino.

## 🎯 Visión General

Este Design System está diseñado específicamente para **Pinteya E-commerce**, siguiendo las mejores prácticas de UX/UI inspiradas en:

- **🛒 Mercado Libre Argentina**: Navegación inferior fija, badges animados, destacados de "Envío gratis"
- **✈️ Airbnb**: Iconografía 3D isométrica, colores vibrantes, estilo friendly
- **📱 Mobile-first**: Optimizado para dispositivos móviles con experiencia desktop mejorada

## 🎨 Principios de Diseño

### 1. **Mobile-First**
- Diseño prioritario para dispositivos móviles
- Navegación táctil optimizada (touch targets 44px+)
- Componentes adaptables y responsive

### 2. **Especialización Pinturería**
- Paleta de colores **Tahiti Gold** (#fc9d04) como color principal
- Iconografía específica del rubro (pinceles, rodillos, baldes)
- Terminología argentina (pesos, envío gratis, etc.)

### 3. **Accesibilidad WCAG 2.1 AA**
- Contraste mínimo 4.5:1 para texto normal
- Focus rings visibles para navegación por teclado
- Screen readers compatibles
- Reduced motion support

### 4. **Performance Optimizado**
- Componentes tree-shaking compatibles
- Animaciones GPU-accelerated
- Bundle size optimizado
- Core Web Vitals optimizados

## 🛠️ Stack Tecnológico

- **Framework**: Next.js 15 + TypeScript
- **Styling**: Tailwind CSS 3.4+
- **Componentes**: shadcn/ui + Radix UI
- **Iconos**: Lucide React
- **Fuentes**: Inter + Euclid Circular A
- **Testing**: Jest + React Testing Library + Playwright

## 🧩 Componentes Principales

### 🔘 **Button**
- **8 variantes**: Primary, Secondary, Outline, Ghost, Destructive, Success, Warning, Link
- **7 tamaños**: sm, md, lg, xl, icon, icon-sm, icon-lg
- **Estados**: Loading, Disabled, con íconos izquierda/derecha
- **Funcionalidades**: Full width, animaciones, accesibilidad

### 🃏 **Card**
- **4 variantes**: Default, Elevated, Outlined, Ghost
- **3 efectos hover**: Lift, Glow, Scale
- **ProductCard especializada**: Para productos de pinturería
- **Componentes**: Header, Content, Footer

### 🏷️ **Badge**
- **12 variantes**: Colores + outline variants
- **Badges especializados**: Descuento, Envío, Stock, Nuevo, Oferta
- **3 tamaños**: sm, md, lg
- **4 animaciones**: Pulse, Bounce, Ping

### 📝 **Input**
- **3 variantes**: Default, Error, Success
- **3 tamaños**: sm, md, lg
- **Funcionalidades**: Labels, helper text, íconos, validaciones
- **Tipos**: Text, Email, Password, Search, Tel, Number

## 🎨 Paleta de Colores

### **Colores Principales**
- **Primary**: Tahiti Gold (#fc9d04) - Color de marca
- **Secondary**: Azul Petróleo (#2c5f5d) - Color secundario
- **Accent**: Tahiti Gold Light (#ffd448) - Acentos

### **Colores Semánticos**
- **Success**: Verde (#22ad5c) - Éxito, envío gratis
- **Error**: Rojo (#f23030) - Errores, descuentos
- **Warning**: Amarillo (#fbbf24) - Advertencias, stock bajo
- **Info**: Azul (#3b82f6) - Información general

### **Colores Neutros**
- **White**: #ffffff
- **Gray Scale**: 50, 100, 200, 300, 400, 500, 600, 900

xl: 1280px  /* Desktop grande */
2xl: 1536px /* Desktop extra grande */

```

</Card>
```
